# Exceptions
## **Теория и практика Java Core. Обработка исключений**

**Цель создания репозитория:** освоение, повторение и закрепление материала по Java Core, практика и разбор учебного кода по теме, в том числе упражнений и лабораторных работ.

**Основная тема:** "Обработка исключений"

**Ключевые подтемы:**
- исключения и иерархия исключений;
- перехват и генерирование исключений;
- использование try-catch, throw, throws, finally;
- встроенные классы и пользовательские подклассы исключений.
